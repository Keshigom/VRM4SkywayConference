# VRM 4 SkyWay Conference


[SkyWay](https://webrtc.ecl.ntt.com/)を利用した
[SkyWay Conference](https://github.com/skyway/skyway-conf)を拡張し、
VRMを用いたビデオ会議機能を追加したものです.
